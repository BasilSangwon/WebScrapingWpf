# WebScrapingWpf
## Summary
* 한번에 검색으로 여러 사이트에서 물건의 가격 및 링크를 알 수 있는 프로젝트
* AngleSharp 패키지를 활용하여 HTML 코드를 읽어서 필요한 태그의 정보를 얻어옴
* MVVM 패턴 활용
* 하나의 View에 ViewModel 2개를 사용 하였으며, 추후 추가 및 삭제가 용이 하게 설계함

## Enviroment
* WPF
* .Net Core 6.0
* C#
* Visual Studio 2022
